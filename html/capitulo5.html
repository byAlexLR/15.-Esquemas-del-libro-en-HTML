5. OPERADORES

Los operadores en JavaScript son símbolos utilizados para realizar operaciones con valores de variables. Se agrupan en varias categorías:

5.1 Operadores Aritméticos

Realizan operaciones matemáticas básicas. Ejemplo:


    let suma = 5 + 3; // Suma
    let multiplicacion = 8 * 2; // Multiplicación
  • +: Suma o concatenación
  • -: Resta
  • *: Multiplicación
  • /: División
  • %: Resto
  • **: Exponenciación
  • ++: Incremento
  • --: Decremento

5.2 Operadores de Asignación

Asignan valores a las variables. Ejemplo:


    let x = 10;
    x += 5; // x ahora es 15
  • =: Asignación simple
  • +=: Asignación con suma
  • -=: Asignación con resta
  • *=: Asignación con multiplicación
  • /=: Asignación con división
  • **=: Asignación con exponenciación
  • %=: Asignación con resto

5.3 Operadores de Comparación

Comparan dos valores y devuelven true o false.


    let esIgual = 5 == "5"; // true
    let esEstricto = 5 === "5"; // false
  • ==: Igualdad
  • ===: Igualdad estricta (valor y tipo)
  • !=: Desigualdad
  • !==: Desigualdad estricta
  • <, <=, >, >=: Comparaciones numéricas

5.4 Operadores Lógicos

Aplican álgebra booleana para evaluar expresiones.


    let and = true && false; // false
    let or = true || false; // true
    let not = !true; // false
  • &&: AND lógico
  • ||: OR lógico
  • !: NOT lógico

5.5 Operador Condicional (Ternario)

Evalúa una condición y devuelve uno de dos valores según el resultado.


    let acceso = edad >= 18 ? "Permitido" : "Denegado";

5.6 Operador typeof

Devuelve el tipo de un valor.


    typeof "Hola"; // "string"

5.7 Precedencia de Operadores

La precedencia determina el orden de ejecución de los operadores. Ejemplo:


    let resultado = 2 + 3 * 4; // 14 (multiplicación primero)

Lista de precedencia de operadores:

  • (): Paréntesis
  • ++, --: Incremento y decremento
  • !, ~, typeof: Negación y tipo
  • **: Exponenciación
  • *, /, %: Multiplicación, división y resto
  • +, -: Adición y sustracción
  • <, <=, >, >=: Comparación
  • ==, !=, ===, !==: Igualdad
  • ?:: Condicional
  • =, +=, -=, *=, /=, %=, **=: Asignación
